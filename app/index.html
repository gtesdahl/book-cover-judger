Methodology

I created this website as my Capstone for the Machine Learning Guild Apprentice Program.

The Book Cover Judger is a CNN Classifier built via ResNet34 transfer learning with the fastai library. The model was trained on book cover images and review data scraped from Goodreads.

The prediction accuracy rate is 44%. While admittedly low, it's still better than random! (33% for classifying into three segments)
